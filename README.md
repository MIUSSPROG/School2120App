# School2120App - Приложение для Школы 2120
## крайняя ветка - account_login

## включает в себя разделы: 
+ Новости
+ Расписание занятий
+ Меню учеников
+ Личный кабинет(Электронный документооборот)
+ Контакты административного персонала

## Стек технологий
+ Clean code
+ MVVM
+ Jetpack navigation component
+ Dagger(Hilt)
+ Room
+ Retrofit
+ Coroutines
+ Flow
+ Glide, Picasso
+ Yandex map API
+ Apache POI
+ Truth
+ Mockito
+ Turbine
